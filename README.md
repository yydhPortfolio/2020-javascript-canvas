# 2020-javascript-canvas
HTML과 JavaScript로 만든 정말 너무 간단한 그림판

<h2>구현된 기능들</h2>
<ul>
  <li>캔버스 위에서 마우스 클릭하면서 드래그시 그리기 가능</li>
  <li>왼쪽밑에 있는 color picker에 선택된 색깔로 그림 그리기가능</li>
</ul>
